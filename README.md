## Taller Multimedial.
### 1. Acercamiento a TouchDesigner.
#### Qué es TouchDesigner? 
##### Es un lenguaje de programación visual basado en nodos para contenido multimedia interactivo en tiempo real . Desarrollado por la empresa "Derivative" de Toronto, es utilizado frecuentemente por artistas, programadores, codificadores creativos, diseñadores de software y artistas para crear performances, instalaciones y obras multimedia fijas.
